## [v1.1.0] - 2026-02-07

### 新增功能
- **服务器管理功能**
  - 添加服务器接口：`POST /api/servers`
  - 获取服务器列表：`GET /api/servers`
  - 获取服务器详情：`GET /api/servers/<id>`
  - 更新服务器信息：`PUT /api/servers/<id>`
  - 删除服务器：`DELETE /api/servers/<id>`
  - 测试服务器连接：`POST /api/servers/<id>/test`

- **数据库存储**
  - SQLite 数据库自动初始化
  - 服务器信息持久化存储
  - 支持服务器状态跟踪（在线/离线/错误）

- **前端页面**
  - 新增服务器管理页面 (`/servers`)
  - 服务器列表展示（卡片式布局）
  - 添加服务器对话框（表单验证）
  - 编辑服务器对话框
  - 服务器连接测试功能
  - 实时刷新机制（每 10 秒）

- **UI/UX 改进**
  - 渐变色服务器卡片设计
  - 模态框表单交互
  - 服务器状态色彩标识
  - 响应式网格布局

### 技术实现
- 后端：Python Flask + SQLite3
- 前端：原生 JavaScript + CSS3
- 通信：REST API + JSON
- 功能：双向数据绑定，表单验证，错误处理
