## 文档模板

新建设计文档请参考以下结构：

```markdown
# {模块/功能名称}

## 概述

简要描述该模块/功能的目的和范围。

## 技术架构

### 组件结构

描述主要组件及其职责。

### 数据流

描述数据如何在组件间流动。

## 接口设计

### API 接口

列出对外暴露的 API。

### 数据模型

描述核心数据结构。

## 设计决策

记录重要的技术决策及其原因。

## 未来规划

计划中的改进和扩展。
```
